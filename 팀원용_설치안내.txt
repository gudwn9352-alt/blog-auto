================================================================
   블로그 원고 이미지 생성 시스템 — 팀원용 설치 안내
================================================================

설치에 필요한 시간: 약 5분
컴퓨터 요구사항: Windows 10/11, 인터넷 연결


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[설치 순서]

1. 관리자에게 받은 폴더를 바탕화면에 복사
   (폴더 안에 "설치및실행.bat" 파일이 있어야 합니다)

2. 관리자에게 ".env.local" 파일을 따로 받기
   → 이 파일을 폴더 안에 넣기

3. "설치및실행.bat" 파일을 더블클릭
   → Node.js, Git 자동 설치 (처음 1회만)
   → 코드 자동 다운로드
   → 패키지 자동 설치
   → 브라우저 자동 열림

4. 끝! 브라우저에서 사용하면 됩니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[프로그램 실행 방법 (2번째부터)]

"설치및실행.bat" 더블클릭 → 자동으로 열림


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[프로그램 업데이트 방법]

관리자가 프로그램을 수정했을 때:
"업데이트.bat" 더블클릭 → 자동으로 최신 버전으로 업데이트


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[프로그램 종료 방법]

검은 창(터미널)을 닫으면 종료됩니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[문제 해결]

"설치및실행.bat 실행이 안 돼요"
→ 파일에서 우클릭 → "관리자 권한으로 실행"

"사이트에 연결할 수 없습니다"
→ 검은 창이 열려있는지 확인. 닫혔으면 다시 더블클릭.

"브랜드가 안 보여요"
→ .env.local 파일이 폴더 안에 있는지 확인
→ 없으면 관리자에게 요청

"이미지가 안 보여요"
→ 이미지는 각 PC에서 따로 생성됩니다.
→ 기존 이미지가 필요하면 관리자에게 요청
